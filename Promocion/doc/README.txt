README Calculadora Simple:
-La calculadora busca los plugins en la carpeta plugins dentro de la carpeta bin, que inicialmente está vacía.
-Si son cargados nuevos plugins en la carpeta plugins, es necesario apretar en el botón "actualizar" para que sean cargados.
-Sólo se pueden ingresar números enteros.
-Una vez que se apretó el botón "calcular" no se podrá volver a realizar una operación hasta apretar el botón "reiniciar".
-Los archivos .CLASS de los plugins implementados se encuentran en la carpeta plugs.
-En la carpeta src se encuentran todos el código fuente de la calculadora y de los plugins implementados.
